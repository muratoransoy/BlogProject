# BlogProject

Murat Oransoy'un kişisel developer blog + portfolio sitesi.

## Stack
- **SSG**: Hugo Extended
- **Tema**: Blowfish (öneri — onay bekliyor)
- **Diller**: English (default) + Türkçe
- **Host**: Cloudflare Pages (free tier)
- **Repo**: GitHub
- **CI**: GitHub Actions (build + link check)
- **Yorum**: giscus
- **Analytics**: Cloudflare Web Analytics
